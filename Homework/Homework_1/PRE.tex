		\section{Question 1}
		
		If $(V,\|\cdot\|)$ is a normed vector space, prove that
		\[
		\|x-y\|\ge \bigl|\|x\|-\|y\|\bigr|\qquad \text{for all }x,y\in V.
		\]
		Using this, prove that if $\|\cdot\|'$ is another norm on $V$ for which there exists a constant $C>0$
		such that
		\[
		\|v\|'\le C\|v\|\qquad \text{for all }v\in V,
		\]
		then the function
		\[
		\|\cdot\|' : (V,\|\cdot\|)\to \mathbb{R}_{\ge 0}
		\]
		is continuous. (This was an important step in the proof that all norms are equivalent in finite
		dimensions we saw in Lecture 1).
		
		\textbf{Answer}
		
		\clearpage
